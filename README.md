## :nut_and_bolt: Development


#### 1. Clone the Repository

```Bash
git clone https://github.com/nightwarriorftw/AnalyticsDashboard.git
cd AnalyticsDashboard
```

#### 2. Install the dependencies

```BASH
yarn install
```

#### 3. Run server:

```BASH
yarn start
```
